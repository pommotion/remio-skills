# 变现方向评估指南

> 被 SKILL.md Phase 5C 引用。评估变现方向时读取此文件。

---

## 8 种变现方式评估

对每首作品，逐一评估适配度（⭐ 不适合 / ⭐⭐ 可以尝试 / ⭐⭐⭐ 高度匹配）：

| # | 变现方式 | 适配度 | 判断依据 |
|---|---------|--------|----------|
| 1 | 流媒体播放分成 | ⭐ | 需要批量作品+长期运营，单首不够 |
| 2 | 短视频 BGM 分成 | | Hook 是否够强？能否剪出 15-30秒？ |
| 3 | 版权授权 | | 是否有明确使用场景（广告/短剧/游戏）？质量是否稳定？ |
| 4 | 定制音乐服务 | | 是否展示了定制能力？歌词是否体现个性化？ |
| 5 | 音效素材包 | | 是否有可提取的转场/冲击/氛围元素？ |
| 6 | 视频创作者配乐 | | 人声是否抢画面？情绪是否匹配视频场景？ |
| 7 | 教程/知识付费 | | 是否有独特方法论？（画面感歌词/汤底公式等） |
| 8 | 工作流/Skill产品 | | 制作流程是否可复用？提示词是否可模板化？ |

## 输出格式

```
💰 变现方向评估

▎最佳变现方向：[Top 1]
  适合原因：[为什么]
  行动建议：[下一步做什么]

▎次选方向：[Top 2]
  适合原因：[为什么]
  行动建议：[下一步做什么]

▎商用就绪度
  当前状态：[草稿 / 可发布 / 可商用]
  还需要：[具体步骤]
  
▎视频创作者视角（核心定位）
  这首歌作为 AI 配乐，适合：
  - 片头音乐：是/否（原因）
  - 情绪 BGM：是/否（原因）
  - 卡点音乐：是/否（原因）
  - 账号主题曲：是/否（原因）
```

## 素材化建议

如果作品适合打包为素材：

```
📦 素材化方案

可拆分片段：
  - 前奏（0:00-0:15）：[适合用途]
  - 主歌片段（0:15-0:45）：[适合用途]
  - 副歌 Hook（0:45-1:00）：[适合用途]
  - 桥段（1:30-1:50）：[适合用途]
  - 尾奏（2:00-2:15）：[适合用途]

可归入素材包：
  □ Vlog BGM 包
  □ 影视氛围配乐包
  □ 短视频转场音效包
  □ 创作者主题曲模板包
```
