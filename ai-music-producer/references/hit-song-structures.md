> 被 SKILL.md Phase 2 引用。创作歌词时读取此文件。

## 一、四种爆款歌词结构

> 数据来源：番茄音乐 AI 爆款热歌生产手册，覆盖华语 Top 热歌的共性特征。

**核心洞察：用反套路的「意象」填入洗脑的「结构」。**

| 模式 | 公式 | 例子 |
|------|------|------|
| **A+A+A+B 排比** | 同一句式叠三遍 + 转折 | 「你听啊XX，你听他XX，只剩我XX」（《冬眠》） |
| **三字连发** | 重复短词三次 | 「滴答滴，滴答滴，滴答滴」「会不会，会不会，会不会」 |
| **签名式** | 「像我这样XX的人」句式 | 「借我十年，借我亡命天涯的勇敢」 |
| **长句引子+排比** | 长句开头 + 三连排比收尾 | 「那些你很冒险的梦，我陪你去疯」→ 后续排比 |

### 使用规则

1. 每首歌的 Hook（副歌核心句）**必须**使用以上至少一种结构
2. 填入的「XX」用 `scene2lyric` 的反套路意象替换，不用套路词（梦想/飞翔/力量/永远）
3. 句式可以微调，但**重复次数不能减**（3 次是最低洗脑线）
4. 至少设计一处「反差反打」——甜美旋律+狠毒意象 / 温柔表达+尖锐内容

### 旋律辅助要素

| 要素 | 说明 | 如何实现 |
|------|------|----------|
| **5(Sol) 音多用** | 适合亚洲人审美，可开心/悲伤/美好/急促 | 歌词自然韵律中多用 Sol 对应的音节 |
| **副歌旋律跨度大** | 爆发力来源于音域跳跃 | Prompt 中用 `--key` + 情绪对比 |
| **三连音节奏** | 面积使用或衔接使用 | Prompt 中加 `triplet feel` / `shuffle rhythm` |

## 二、洗脑度自检清单

在 Checkpoint 2 之前，逐项确认：

```
□ 副歌有重复结构（A+A+A+B / 三字连发 / 签名式 / 排比）
□ 重复部分至少出现 3 次
□ 填入的是反套路意象（非「梦想」「飞翔」「力量」）
□ 副歌旋律跨度足够大（可通过 Prompt 中 --key 和情绪推断）
□ 有至少一处「反差反打」设计
```

**5 项中至少通过 3 项才能进入 Phase 3。** 不足 3 项 → 回到歌词创作补充。

### 洗脑度评分（用于 Phase 5 六维诊断第 6 维）

| 分数 | 标准 |
|------|------|
| 10 | 副歌有明确重复结构，且填入反套路意象，一听上头 |
| 7-9 | 有重复但不极致，或意象稍套路 |
| 4-6 | 副歌无重复结构，旋律平 |
| 1-3 | 整首歌无记忆点 |

## 三、歌词题材偏好（番茄平台验证）

| 题材 | 关键词 | 占比 |
|------|--------|------|
| 爱情&情绪（最多） | 放不下、算了吧、我真的累了 | ~60% |
| 励志&成长 | 我一定可以、坚持 | ~20% |
| 家乡&故乡 | 妈妈、老屋、村口、月光 | ~10% |
| **反差反打**（高转化新趋势） | 甜美旋律+狠毒歌词 / 温柔女声唱狠话 | ↑ |

### 番茄平台歌词偏好

| 要 | 不要 |
|-----|------|
| 有画面感，有细节 | 堆砌词藻 |
| 直白，无生僻词 | 复杂修辞 |
| 年轻人听感 | 老派表达 |
