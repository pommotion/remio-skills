# Music Prompt Templates

按「流派 × 场景」矩阵提供现成的 AI 音乐 Prompt 模板。每个模板包含歌词汤底 + 旋律汤底 + 进阶技巧，可直接复制到 Suno / Udio / 豆包使用。

## 适用场景

- 用户说"帮我写一首 XX 风格的歌"
- 用户要做短视频 BGM / 广告歌 / 情歌 / 励志歌 / 国风歌
- 用户有主题但不知道怎么组织 prompt
- 用户想快速生成特定流派的音乐

## 核心原则

1. **汤底 = 方法论压缩包**：每个模板已内置经典歌曲结构经验、音乐心理学、编曲动态
2. **只填空，不从头写**：用户只需替换 `[方括号]` 中的变量
3. **多次生成 > 精调 prompt**：同一模板跑 3-5 次，选最好的

## 模板正文

### T1. 流行 - 短视频 BGM

**适用**: 15-30 秒副歌片段，抖音/快手/视频号

#### 歌词汤底

```
请写一首现代流行风格的歌曲，主题是[主题]。

只需要副歌部分（Chorus），时长约15-30秒。

副歌：核心 Hook，重复"[核心句]"2-3次
要求：
- Hook 不超过9个字，包含至少一个重复字
- 用具体物象代替抽象情感（"冰箱里还有你买的药" > "我好想你"）
- 结尾留一个画面余韵
```

#### 旋律汤底

```
请生成一首现代流行风格的短片段（仅副歌），要求如下：

风格：现代流行 / Dance Pop
BPM：120-128
调式：C大调 或 G大调
时长：15-30秒
结构：副歌 → 尾奏
乐器编配：
  副歌：电子鼓点 + 合成器 Pad + 钢琴
  尾奏：渐弱
情感基调：[欢快/甜蜜/感伤/热血]
人声类型：[女声/男声]，清晰
```

#### 进阶技巧

- 指定参考：`参考[歌名]的节奏感`（如《孤勇者》《起风了》）
- 反例：`不要口水歌风格，不要网络神曲`
- 加钩子：`副歌第一个字用动词或感叹词开头`

### T2. 流行 - 广告歌 / 品牌主题曲

**适用**: 品牌主题曲、门店洗脑歌、产品广告配乐

#### 歌词汤底

```
请写一首流行风格的[品牌/产品]广告歌，主题是[核心卖点]。

结构：
前奏（4秒）：品牌名或核心 Slogan
主歌1：描述用户痛点场景
副歌：核心 Slogan 重复 3 次 + 产品卖点
主歌2：使用后的变化
副歌：重复
尾奏（4秒）：品牌名收尾

要求：
- 歌词口语化，像朋友推荐
- 副歌 Slogan 不超过7个字
- 不要"奢华""尊享"等广告腔
- 用日常场景代替功能描述
```

#### 旋律汤底

```
请生成一首流行广告歌，要求如下：

风格：流行 / 轻快 Pop
BPM：110-120
调式：C大调 或 F大调
结构：前奏-主歌A-副歌-主歌B-副歌-尾奏
乐器编配：
  前奏：明亮的钢琴或吉他
  主歌：轻拍鼓 + 贝斯
  副歌：加入合成器 + 拍手
情感基调：积极、温暖、信任感
人声类型：[女声/男声]，亲切自然
总时长：60-90秒
```

#### 进阶技巧

- 门店洗脑歌：`副歌用重复旋律，听2遍就能哼`
- 香水/美妆品牌：`加入环境音暗示（风铃/水声）`
- 餐饮品牌：`歌词中出现"味道""香气""热气腾腾"等感官词`

### T3. 流行 - 情歌

**适用**: 失恋、思念、暗恋、甜蜜

#### 歌词汤底

```
请写一首流行情歌，主题是[失恋/思念/暗恋/甜蜜]。

结构：
主歌A：用一个具体场景开头（"便利店玻璃窗上的雾气"）
主歌B：用[具体物件]回忆[过往]
预副歌：情绪转折，从[情感A]到[情感B]
副歌：核心 Hook，重复"[关键句]"3次以上
主歌C：新视角，[另一个画面/故事]
副歌：重复
桥段：转变节奏，用[比喻/意象]升华
副歌：最后一次，最强烈
尾奏：留一个[意象]结束

反套路检查：
- 禁止出现"泪水""心碎""想你""永远"等陈词滥调
- 每段至少一个具体物象
- 情绪用场景暗示，不用形容词直说
```

#### 旋律汤底

```
请生成一首流行情歌，要求如下：

风格：抒情流行 / Ballad
BPM：68-80
调式：[A小调（忧伤）/ D大调（甜蜜）]
结构：前奏-主歌A-主歌B-预副歌-副歌-主歌C-副歌-桥段-副歌-尾奏
乐器编配：
  前奏：钢琴独奏
  主歌：钢琴 + 轻弦乐
  预副歌：加入吉他
  副歌：弦乐 + 鼓点渐强
  桥段：只留人声 + 钢琴
  尾奏：钢琴渐弱
情感基调：[深沉忧伤 / 温暖甜蜜 / 微酸思念]
人声类型：[女声，清澈略沙哑 / 男声，温暖低沉]
```

#### 进阶技巧

- 失恋歌：`主歌用过去时，副歌用现在时——制造「回不去了」的时差感`
- 思念歌：`桥段用电话未接通的音效暗示`
- 甜蜜歌：`BPM 可以提到 85-95，加入轻快吉他`

### T4. 流行 - 励志歌

**适用**: 正能量、奋斗、突破、自我成长

#### 歌词汤底

```
请写一首流行励志歌，主题是[逆境/突破/成长/自我]。

结构：
主歌A：描述低谷状态（用场景，不说"我很惨"）
主歌B：转折点——一个[具体物件/事件]触发了改变
预副歌：蓄力，节奏加快
副歌：核心宣言，重复"[宣言句]"3次以上
主歌C：行动中的画面
副歌：重复
桥段：反思 + 再出发
副歌：最后爆发
尾奏：余韵画面

要求：
- 宣言句不超过9字，用肯定句
- 低谷描写不超过2段，快速转向行动
- 禁止"相信自己""永不放弃"等口号
- 用具体行动画面代替抽象鼓励
```

#### 旋律汤底

```
请生成一首流行励志歌，要求如下：

风格：Pop Rock / Arena Pop
BPM：120-135
调式：E小调（主歌压抑）→ E大调（副歌爆发）
结构：前奏-主歌A-主歌B-预副歌-副歌-主歌C-副歌-桥段-副歌-尾奏
乐器编配：
  前奏：安静吉他或钢琴
  主歌：贝斯 + 轻鼓
  预副歌：鼓点加密
  副歌：全乐器爆发（吉他+鼓+弦乐+合成器）
  桥段：减到人声+钢琴，再蓄力
情感基调：从压抑到爆发
人声类型：[男声/女声]，有力量感
```

### T5. 国风 - 情歌/叙事

**适用**: 古风情歌、江湖叙事、仙侠意境

#### 歌词汤底

```
请写一首国风流行歌曲，主题是[相思/离别/江湖/仙侠]。

结构：
主歌A：用[自然意象]开篇（月/风/柳/雁/梅/雪/山/雾）
主歌B：叙事推进，用[古典物象]（铜镜/玉佩/书简/青灯/纸伞）
预副歌：情绪转折
副歌：核心 Hook，用五声音阶意象
主歌C：新视角
副歌：重复
桥段：用[典故/诗词化用]升华
副歌：最后爆发
尾奏：留一个自然意象结束

要求：
- 歌词用半文半白风格
- 副歌 Hook 含至少一个自然意象
- 禁止现代口语（"然后""其实""因为"）
- 韵脚统一，每段押同韵
```

#### 旋律汤底

```
请生成一首国风流行歌曲，要求如下：

风格：国风流行 / 古风
BPM：72-88
调式：D小调五声 或 宫调式
结构：前奏-主歌A-主歌B-预副歌-副歌-主歌C-副歌-桥段-副歌-尾奏
乐器编配：
  前奏：古筝 + 竹笛
  主歌：吉他 + 轻鼓点
  预副歌：加入二胡
  副歌：弦乐 + 二胡 + 鼓
  桥段：只留人声 + 古筝
  尾奏：竹笛渐远
情感基调：[空灵思念 / 豪迈江湖 / 清冷孤寂]
人声类型：[女声，清透 / 男声，温润]
```

#### 进阶技巧

- 化用诗词：`副歌可化用一句[古诗]，但不是直接引用，而是意境化用`
- 仙侠感：`BPM 提到 90-100，副歌加入电子合成器制造「穿越」感`
- 武侠感：`加入鼓点 + 琵琶，BPM 100-115`

### T6. 国风 - 广告歌

**适用**: 茶叶、酒、文旅、中式美妆等品牌

#### 歌词汤底

```
请写一首国风品牌广告歌，品牌是[品牌名]，核心卖点是[卖点]。

结构：
前奏（4秒）：品牌名 + 核心意象
主歌：用[传统场景]（茶馆/酒楼/园林/山间）描述品牌氛围
副歌：品牌 Slogan + 核心卖点，重复 2 次
桥段：从古到今的意境过渡
副歌：最后一次
尾奏（4秒）：品牌名收尾

要求：
- Slogan 不超过7字
- 歌词含至少3个感官描写（视觉+嗅觉+触觉优先）
- 用「品」「闻」「触」等动词引导感官体验
```

#### 旋律汤底

```
请生成一首国风广告歌，要求如下：

风格：新中式流行
BPM：85-100
调式：G大调五声
结构：前奏-主歌-副歌-桥段-副歌-尾奏
乐器编配：
  前奏：古筝 + 琵琶
  主歌：吉他 + 轻鼓
  副歌：加入弦乐 + 笛子
情感基调：雅致、从容、高级感
人声类型：女声，温婉
总时长：60-90秒
```

### T7. R&B - 情歌

**适用**: 都市情感、深夜氛围、微醺感

#### 歌词汤底

```
请写一首 R&B 风格的情歌，主题是[暧昧/遗憾/深夜思念]。

结构：
Intro：环境音暗示（雨声/深夜/酒杯）
Verse 1：深夜场景，用[光影细节]营造氛围
Pre-Chorus：情绪蓄力
Chorus：Hook，半说半唱
Verse 2：回忆闪回
Chorus：重复
Bridge：空灵段落，用[比喻]升华
Chorus：最后爆发
Outro：环境音淡出

要求：
- 歌词用都市口语，略慵懒
- Hook 用重复的短句，制造律动感
- 禁止"永远""一生一世"等过于郑重的词
- 场景用夜晚、室内、半私密空间为主
```

#### 旋律汤底

```
请生成一首 R&B 情歌，要求如下：

风格：Contemporary R&B / Neo-Soul
BPM：75-90
调式：Bb小调 或 G小调
结构：Intro-Verse1-PreChorus-Chorus-Verse2-Chorus-Bridge-Chorus-Outro
乐器编配：
  Intro：电钢琴 + 环境音
  Verse：电钢琴 + 轻拍鼓 + 贝斯
  Chorus：加入合成器 + 和声
  Bridge：减到人声 + 电钢琴
情感基调：慵懒、微醺、暧昧
人声类型：[女声，气声 / 男声，R&B 唱腔]
```

### T8. 嘻哈 - 叙事

**适用**: 个人故事、社会议题、态度表达

#### 歌词汤底

```
请写一首中文说唱歌曲，主题是[主题]。

结构：
Intro：态度宣言或场景设定
Verse 1：叙事主线，[具体故事/场景]
Hook：核心态度，重复 2 次，不超过8字
Verse 2：新论据或视角转换
Hook：重复
Bridge：旋律化段落，节奏放慢
Verse 3：总结 + 升华
Hook：最后一次
Outro：态度收尾

要求：
- Verse 用叙事而非说教，每段至少1个具体故事
- Hook 用短句+重复，容易被记住
- 韵脚密度：每4拍至少2个韵
- 禁止空洞口号
```

#### 旋律汤底

```
请生成一首中文说唱歌曲，要求如下：

风格：[Boom Bap / Trap / Melodic Rap]
BPM：80-95（Boom Bap）或 130-145（Trap）
调式：C小调 或 A小调
结构：Intro-Verse1-Hook-Verse2-Hook-Bridge-Verse3-Hook-Outro
乐器编配：
  Intro：采样或环境音
  Verse：鼓机 + 贝斯 + 采样
  Hook：加入旋律合成器
  Bridge：减到钢琴或吉他
情感基调：[态度坚定 / 反思深沉 / 热血]
人声类型：男声/女声，说唱腔
```

### T9. Lo-Fi - 氛围

**适用**: 学习 BGM、工作背景、睡前放松、直播背景

#### 歌词汤底

```
（Lo-Fi 通常无歌词或极少歌词，以下是可选的人声片段）

如需人声：
- 只在 Hook 部分加 1-2 句轻柔哼唱
- 歌词不超过 10 个字
- 内容为简短的情绪短语（"rain keeps falling" / "still here"）
- 女声气声，效果器：reverb + delay
```

#### 旋律汤底

```
请生成一首 Lo-Fi Hip Hop 氛围曲，要求如下：

风格：Lo-Fi Hip Hop / Chill Hop
BPM：70-85
调式：D小调 或 F大调
结构：循环段落，无传统歌曲结构
时长：2-4分钟
乐器编配：
  主体：电钢琴或吉他循环
  节奏：轻鼓机 + 黑胶采样杂音（vinyl crackle）
  氛围：环境采样（雨声/咖啡馆/蝉鸣）
  低频：温暖的贝斯线
情感基调：放松、怀旧、舒适
人声类型：无 或 极少气声
```

#### 进阶技巧

- 加环境音：`背景加入[雨声/海浪/咖啡馆嘈杂/火车]采样`
- 黑胶质感：`全曲加入 vinyl crackle 效果`
- 学习专注：`避免突兀变化，保持一致的音量和节奏`

## 使用方式

### 直接使用

用户说"帮我写一首 XX 歌"时：

1. 从矩阵中匹配最接近的模板
2. 展示模板，让用户确认 `[方括号]` 中的变量
3. 用户填好后，组合歌词汤底 + 旋律汤底为完整 prompt
4. 用户复制到 Suno/Udio/豆包 生成

### 与其他 skill 联动

- **scene2lyric**: 先用 scene2lyric 从场景辞海提取画面感素材，再用本模板组装
- **lyric-qa**: 生成歌词后用 lyric-qa 检测质量
- **ai-music-producer**: 全流程 skill 会在对应 Phase 调用本模板

### 自定义扩展

用户可以基于模板衍生新流派：

```
请基于 T3 流行情歌模板，改写为 Bossa Nova 版本：
- BPM 调整为 100-120
- 乐器改为：尼龙弦吉他 + 轻鼓刷 + 手风琴
- 歌词用"慵懒午后"氛围替代"深夜忧伤"
```

## 保存到知识库

当用户说"保存这个模板"或"加入我的模板库"时：

1. `search_notes` 检查是否已有同类模板
2. 用 `create_note` 保存，标题格式：`🎵 音乐模板 - [流派]-[场景]`
3. 加入 `Prompt Library` 集合

## 质量规则

- 每个模板必须同时包含歌词汤底和旋律汤底
- 所有 `[方括号]` 变量必须有意义明确的提示文字
- 旋律汤底的 BPM 和调式必须符合该流派的典型范围
- 进阶技巧不得引入模板基础版中没有的新结构
- 模板之间保持一致的格式（结构 → 汤底 → 进阶）
